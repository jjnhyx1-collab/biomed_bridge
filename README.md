# BioMed Bridge

의생명공학 진로 포트폴리오 웹사이트입니다. GitHub Pages로 바로 배포할 수 있도록 구성했습니다.

## 이번 업데이트 내용

- 메뉴 추가: 진로 로드맵, 논문 읽기, 참고문헌, 느낀 점
- 탐구 주제 추가: 기존 3개에서 6개로 확장
- 논문·참고문헌 섹션 강화: 논문 읽기 체크리스트, 검색 키워드, 국문·영문 자료, 이미지 출처 원칙 추가

## 파일 구성

- index.html: 사이트의 구조와 내용
- style.css: 사이트 디자인
- README.md: 프로젝트 설명

## GitHub Pages 업데이트 방법

1. 압축 파일을 풉니다.
2. 기존 GitHub 저장소로 들어갑니다.
3. 기존 index.html, style.css, README.md를 새 파일로 덮어씁니다.
4. Commit changes를 클릭합니다.
5. GitHub Pages 주소에 접속해 새로고침합니다.
6. 반영이 늦으면 1~3분 뒤 다시 새로고침합니다.
